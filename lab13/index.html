Лабораторна робота 13 - Об'єктна модель управління курсами

Інструкції:

Ця сторінка демонструє реалізацію об'єктної моделі для системи управління курсами з використанням прототипного наслідування в JavaScript.

Модель включає:

  • Курс (назва, список студентів, методи для додавання студентів)
  • Викладач (ім'я, список курсів, методи для додавання курсів)
  • Студент (ім'я, список курсів, метод перегляду курсів)

Прототипне наслідування використовується для спільних методів.

Результати:

Відкрийте консоль розробника (F12), щоб побачити демонстрацію об'єктної моделі.

Примітка: Ця реалізація демонструє концепції:

  • Створення об'єктів у JavaScript
  • Функції-конструктори
  • Прототипне наслідування
  • Реалізація методів
  • Зв'язки між об'єктами
